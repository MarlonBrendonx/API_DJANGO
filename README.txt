1-API-DJANGO

Na primeira API implementada, foi utilizado os métodos padrão do rest_framework. Dessa forma, temos as seguintes rotas padrão:

GET    http://localhost:8000/user/ 
POST   http://localhost:8000/user/      
PUT    http://localhost:8000/user/      
DELETE http://localhost:8000/user/      
GET    http://localhost:8000/export/ -> Para exportar os usuários em formato xls
